# Quran Gift Shop - Premium Islamic E-commerce

A full-stack MERN e-commerce application for an Islamic brand, featuring a premium design, smooth animations, and comprehensive SEO optimization.

## Features
- **Elegant UI/UX**: Custom theme (Deep Navy, Warm Orange, Soft Beige).
- **Smooth Animations**: Framer Motion transitions and hover effects.
- **Authentication**: Secure JWT-based login and registration.
- **Product Management**: Filterable product listing by categories.
- **Cart System**: Fully functional shopping cart with persistent storage.
- **Checkout**: Multi-step checkout with COD and Mock Payment options.
- **Admin Panel**: Dashboard for managing products, orders, and users.
- **SEO Ready**: Semantic HTML, JSON-LD Schema, and dynamic meta tags.

## Tech Stack
- **Frontend**: React (Vite), Tailwind CSS, Framer Motion, Lucide React.
- **Backend**: Node.js, Express, MongoDB, JWT.
- **Tools**: Axios, React Router, React Hot Toast, React Helmet.

## Prerequisites
- Node.js installed.
- MongoDB installed and running locally.

## Installation & Running

### 1. Clone the repository
```bash
git clone <repository-url>
cd quran-gift-shop
```

### 2. Setup Backend
```bash
cd server
npm install
# Create a .env file based on .env.example
npm start
```

### 3. Setup Frontend
```bash
cd client
npm install
npm run dev
```

## Folder Structure
- `client/`: Frontend React application.
- `server/`: Backend Express API.
- `server/models/`: Database schemas.
- `client/src/components/`: Reusable UI components.
- `client/src/pages/`: Main application pages.
